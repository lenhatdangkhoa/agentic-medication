# BIBM Formatting Notes

## Current Assumption

I could not find a BIBM 2026-specific author-instructions page during this pass, so the manuscript is formatted as a standard IEEE conference paper using `IEEEtran` conference mode.

- IEEE’s official conference manuscript templates remain the safe default when a venue-specific page is unavailable.
