# Overleaf Bundle

## Main Files

- `main.tex`: IEEE-style conference manuscript source
- `refs.bib`: BibTeX bibliography seed

## Supporting Docs

The `docs/` folder contains working notes that may help while editing in Overleaf:

- paper draft in markdown
- paper outline
- verified core references
- figure and table plan
- submission checklist
- formatting notes

## Notes

- This bundle assumes standard IEEE conference formatting via `IEEEtran`.
- Figures are not rendered yet; the manuscript currently compiles without external figure files.
- Before final submission, verify the exact BIBM 2026 author instructions.
